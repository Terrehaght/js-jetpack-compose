New Customization Features 1. Tab Styles 'pill' - Rounded pill tabs like your screenshot (with optional animated indicator) 'underline' - Classic underline tabs 'segmented' - iOS-style segmented control 'icon-pill' - Icon-only circular pill tabs 'default' - Original Material Design style 2. Theming Dark/Light themes - Automatic color scheme switching Custom accent colors - Change the primary color dynamically CSS custom properties - Full color customization support 3. Animation Options 'slide' - Slide transitions (default) 'fade' - Fade in/out 'scale' - Scale effect 4. Enhanced Methods setTabStyle(style) - Change tab style on the fly setAccentColor(color) - Update accent color setTheme('light'|'dark') - Toggle theme setAnimation(type) - Change animation style Usage Example const tabs = AndroidComponents.TabLayout({ tabStyle: 'pill', // Use pill style animatedIndicator: true, // Sliding pill indicator theme: 'dark', // Dark theme accentColor: '#8b5cf6', // Purple accent animation: 'fade', // Fade animation mode: 'fixed', tabs: [ { icon: 'fas fa-home', text: 'Home' }, { icon: 'fas fa-user', text: 'Profile' }, { icon: 'fas fa-bell', text: 'Alerts', badge: '3' } ], pages: [ document.createElement('div'), // Your content document.createElement('div'), document.createElement('div') ] }); // Change style dynamically tabs.setTabStyle('underline'); tabs.setAccentColor('#ec4899'); tabs.setTheme('light');
