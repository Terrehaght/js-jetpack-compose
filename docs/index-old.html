Getting Started

Installation

Add the library scripts to your HTML file in the correct order. Each module builds upon the previous ones.

Dependency Order Matters

Make sure to include layouts.js first, followed by material-views.js, then other modules. Also include BoxIcons for icon support.

1

Include BoxIcons (Required for icons)

The library uses BoxIcons for all icon components.

HTML
<link href="https://unpkg.com/[email protected]/css/boxicons.min.css" rel="stylesheet">
2

Include Core Library Scripts

Add the library modules in the correct dependency order:

HTML
<!-- Core Layout System (REQUIRED FIRST) --> <script src="https://cdn.jsdelivr.net/gh/Terrehaght/js-jetpack-compose@main/js-jetpack-compose/layouts.js"></script> <!-- Material UI Components (REQUIRED SECOND) --> <script src="https://cdn.jsdelivr.net/gh/Terrehaught/js-jetpack-compose@main/js-jetpack-compose/material-views.js"></script> <!-- Dialogs & Snackbars --> <script src="https://cdn.jsdelivr.net/gh/Terrehaught/js-jetpack-compose@main/js-jetpack-compose/material-notifiers.js"></script> <!-- Bottom Navigation --> <script src="https://cdn.jsdelivr.net/gh/Terrehaught/js-jetpack-compose@main/js-jetpack-compose/bottom_nav.js"></script> <!-- Navigation Drawer & Toolbar --> <script src="https://cdn.jsdelivr.net/gh/Terrehaught/js-jetpack-compose@main/js-jetpack-compose/drawer_tools.js"></script> <!-- TabLayout & ViewPager --> <script src="https://cdn.jsdelivr.net/gh/Terrehaught/js-jetpack-compose@main/js-jetpack-compose/tabs.js"></script> <!-- RecyclerView --> <script src="https://cdn.jsdelivr.net/gh/Terrehaught/js-jetpack-compose@main/js-jetpack-compose/recycler_view.js"></script>

Layout System

Layouts

Powerful layout containers for organizing your UI components using Android-style patterns.

Column / LinearLayout

Arrange children vertically with flexible spacing, gravity, and weight distribution.

Row

Arrange children horizontally with alignment and gap controls.

GridLayout

CSS Grid-based layout with column/row spanning support.

FrameLayout

Stack children on top of each other with gravity positioning.

ConstraintLayout

Absolute positioning with constraint-based relationships.

ScrollView

Scrollable container for content that exceeds viewport.

Column / Row / LinearLayout Layout

The fundamental layout containers. Column arranges children vertically, Row arranges horizontally.

JavaScript
const { Column, Row, LinearLayout } = AndroidLayouts; // Vertical layout const myColumn = Column({ padding: 16, gap: 12, gravity: 'center', width: 'match_parent', background: '#f5f5f5' }, [ TextView({ text: 'Hello' }), TextView({ text: 'World' }) ]); // Horizontal layout const myRow = Row({ gap: 8, gravity: 'space_between' }, [child1, child2]); // Mount to DOM AndroidLayouts.mount(myColumn, 'app');

Properties

Property Type Default Description
orientation 'vertical' | 'horizontal' 'vertical' Layout direction
gravity string undefined Alignment: 'center', 'start', 'end', 'space_between', 'space_around', 'space_evenly'
gap number 0 Spacing between children in pixels
padding number | object undefined Internal padding. Can be number or {top, right, bottom, left, horizontal, vertical, all}
width 'match_parent' | 'wrap_content' | number undefined Width constraint
height 'match_parent' | 'wrap_content' | number undefined Height constraint
wrap boolean false Enable flex-wrap for Row layouts

Methods

getElement() Returns the underlying HTMLElement
addChild(child) Dynamically add a child component
removeChild(child) Remove a child component
clear() Remove all children
show() / hide() / invisible() Visibility controls (visible, gone, invisible)

const { RecyclerView, Card, Row, Column, TextView, Avatar } = AndroidComponents; // Define your adapter (item renderer) const userAdapter = (user, index) => { return Card({ margin: 8, padding: 16 }, [ Row({ gap: 16, gravity: 'center' }, [ Avatar({ text: user.name }), Column({}, [ TextView({ text: user.name, textStyle: 'bold' }), TextView({ text: user.email, textSize: 12 }) ]) ]) ]); }; // Create RecyclerView const userList = RecyclerView({ data: users, adapter: userAdapter, layoutManager: 'linear', height: '100vh', divider: true, gap: 8, emptyText: 'No users found', emptyIcon: 'bx bx-user-x', onLoadMore: async (offset) => { const newUsers = await fetchUsers(offset); return { data: newUsers, hasMore: newUsers.length > 0 }; } });

dataarrayArray of data items

layoutManager'linear' | 'grid'Layout type
gridColumnsnumberColumns for grid layout
dividerbooleanShow dividers between items
gapnumberSpacing between items
heightnumber | stringContainer height

Optimized Methods (No Full Re-render)

addItem(item) Add item at end (optimized)
addItemAt(index, item) Add item at specific index (optimized)
updateItem(index, item) Update single item (optimized)
removeItem(index) Remove single item (optimized)

Batch Operations

addData(items) Add multiple items at once
updateItemRange(startIndex, items) Update range of items
removeItemRange(startIndex, count) Remove range of items
setData(items) Replace entire dataset (full re-render)
clear() Remove all items

Android-Style Notify Methods

notifyDataSetChanged() Full dataset changed (re-render)
notifyItemInserted(index) Single item inserted
notifyItemRemoved(index) Single item removed
notifyItemChanged(index) Single item changed
notifyItemRangeInserted(index, count) Range of items inserted

Data Access & Scroll Methods

getData() / getItem(index) / getItemCount() Access list data
scrollToTop() / scrollToBottom() / scrollToIndex(index) Scroll control
setHasMore(bool) / loadMore() / isLoading() Infinite scroll control

Real-time Updates Example

JavaScript
// Real-time updates without re-rendering socket.on('user:added', (user) => { userList.addItem(user); // Optimized, no re-render }); socket.on('user:updated', (user) => { const index = userList.getData().findIndex(u => u.id === user.id); if (index >= 0) { userList.updateItem(index, user); // Optimized } }); socket.on('user:removed', (userId) => { const index = userList.getData().findIndex(u => u.id === userId); if (index >= 0) { userList.removeItem(index); // Optimized } });
