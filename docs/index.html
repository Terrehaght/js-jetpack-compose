JS Jetpack Compose

A comprehensive JavaScript UI framework that brings Android's Jetpack Compose paradigm to the web. Build beautiful, responsive applications with familiar Android-like layouts and Material Design components.

Installation

To use JS Jetpack Compose, include all required libraries in your HTML file in the correct order:

Required Scripts All Required

<!-- Core Layouts -->
<script src="https://cdn.jsdelivr.net/gh/Terrehaght/js-jetpack-compose@main/js-jetpack-compose/layouts.js"></script>

<!-- Android Framework -->
<script src="https://cdn.jsdelivr.net/gh/Terrehaght/js-jetpack-compose@main/js-jetpack-compose/AndroidFramework.js"></script>

<!-- Material Components -->
<script src="https://cdn.jsdelivr.net/gh/Terrehaght/js-jetpack-compose@main/js-jetpack-compose/material-views.js"></script>

<!-- Material Notifiers -->
<script src="https://cdn.jsdelivr.net/gh/Terrehaght/js-jetpack-compose@main/js-jetpack-compose/material-notifiers.js"></script>

<!-- Main Activity -->
<script src="https://cdn.jsdelivr.net/gh/Terrehaght/js-jetpack-compose@main/js-jetpack-compose/.Mainactivity.js"></script>

<!-- Bottom Navigation -->
<script src="https://cdn.jsdelivr.net/gh/Terrehaght/js-jetpack-compose@main/js-jetpack-compose/bottom_nav.js"></script>

<!-- Drawer Tools -->
<script src="https://cdn.jsdelivr.net/gh/Terrehaght/js-jetpack-compose@main/js-jetpack-compose/drawer_tools.js"></script>

<!-- Tabs -->
<script src="https://cdn.jsdelivr.net/gh/Terrehaght/js-jetpack-compose@main/js-jetpack-compose/tabs.js"></script>

<!-- RecyclerView -->
<script src="https://cdn.jsdelivr.net/gh/Terrehaght/js-jetpack-compose@main/js-jetpack-compose/recycler_view.js"></script>

<!-- Boxicons (for icons) -->
<link href="https://unpkg.com/boxicons@2.1.4/css/boxicons.min.css" rel="stylesheet">
Important: Load the scripts in the exact order shown above. Some libraries depend on others being loaded first (e.g., material-views.js requires layouts.js).

Quick Start Example

Here's a complete example showing how to build a simple app with login and signup forms:

Complete Example

const loginForm = Column({
    gap: 16,
    padding: 24
}, [
    MaterialInput({
        label: 'Email',
        type: 'email',
        leadingIcon: 'bx bx-envelope'
    }),
    MaterialInput({
        label: 'Password',
        type: 'password',
        leadingIcon: 'bx bx-lock'
    }),
    MaterialButton({
        text: 'Login',
        variant: 'filled',
        fullWidth: true
    })
]);

const signupForm = Column({
    gap: 16,
    padding: 24
}, [
    MaterialInput({
        label: 'Full Name',
        leadingIcon: 'bx bx-user'
    }),
    MaterialInput({
        label: 'Email',
        type: 'email',
        leadingIcon: 'bx bx-envelope'
    }),
    MaterialInput({
        label: 'Password',
        type: 'password',
        leadingIcon: 'bx bx-lock'
    }),
    MaterialButton({
        text: 'Sign Up',
        variant: 'filled',
        fullWidth: true
    })
]);

const app = ScrollView({
    width: 'match_parent',
    height: 'match_parent'
}, [
    Column({
        width: 'match_parent',
        padding: 24,
        gap: 24,
        background: '#ffffff'
    }, [
        Column({
            gap: 4,
            margin: { top: 32 }
        }, [
            TextView({
                text: 'Welcome',
                textSize: 36,
                textStyle: 'bold',
                textColor: '#1976d2',
                gravity: 'center'
            }),
            TextView({
                text: 'Please login or create an account',
                textSize: 14,
                textColor: '#666',
                gravity: 'center'
            })
        ]),

        TabLayout({
            tabStyle: 'pill',
            animation: 'fade',
            tabs: [
                { text: 'Login', icon: 'bx bx-log-in' },
                { text: 'Register', icon: 'bx bx-user-plus' }
            ],
            pages: [loginForm, signupForm],
            mode: 'fixed',
            swipeable: false
        })
    ])
]);

AndroidLayouts.mount(app, 'app');

LinearLayout

A flexible layout container that arranges children in a single direction (vertical or horizontal), similar to Android's LinearLayout.

Usage

LinearLayout(props = {}, children = [])

Properties

Property Type Default Description
orientation string 'vertical' 'vertical' | 'horizontal' - Direction of child layout
gravity string - center | start | end | center_horizontal | center_vertical | space_between | space_around | space_evenly
padding number/object - Padding as number or object with all/vertical/horizontal/top/right/bottom/left
margin number/string - Margin value
background string - CSS background value
gap number/string 0 Space between children
width string/number - 'match_parent' | 'wrap_content' | number/string value
height string/number - 'match_parent' | 'wrap_content' | number/string value
wrap boolean false Enable flex wrap

Methods

  • getElement() → HTMLElement
  • addChild(child)
  • removeChild(child)
  • clear()
  • setVisibility('visible' | 'invisible' | 'gone')
  • show()
  • hide()
  • invisible()
  • isVisible() → boolean

Example

const layout = LinearLayout({
    orientation: 'vertical',
    gravity: 'center',
    gap: 16,
    padding: 24,
    background: '#f5f5f5'
}, [
    TextView({ text: 'Hello World' }),
    MaterialButton({ text: 'Click Me' })
]);

Column & Row

Convenient shorthand components for vertical and horizontal LinearLayouts.

Column

Vertical LinearLayout shorthand.

Column(props, children) === LinearLayout({ ...props, orientation: 'vertical' }, children)

Row

Horizontal LinearLayout shorthand.

Row(props, children) === LinearLayout({ ...props, orientation: 'horizontal' }, children)

Example

// Vertical layout
const form = Column({
    gap: 16,
    padding: 24
}, [
    MaterialInput({ label: 'Name' }),
    MaterialInput({ label: 'Email' }),
    MaterialButton({ text: 'Submit' })
]);

// Horizontal layout
const header = Row({
    gap: 12,
    padding: 16
}, [
    Icon({ name: 'bx bx-menu' }),
    TextView({ text: 'App Title' })
]);

GridLayout

A CSS Grid-based container for creating grid layouts.

Usage

GridLayout(props = {}, children = [])

Properties

Property Type Default Description
columns number 2 Number of columns
rows number - Number of rows (optional)
gap number/string 0 Gap between items
columnGap number/string - Override column gap
rowGap number/string - Override row gap
autoFlow string 'row' 'row' | 'column' | 'dense'

Grid Parameters (for children)

Use GridParams(params, child) to set grid-specific positioning:

Parameter Type Description
columnSpan number Number of columns to span
rowSpan number Number of rows to span
column number Starting column position
row number Starting row position

Example

const grid = GridLayout({
    columns: 3,
    gap: 16,
    padding: 24
}, [
    Card({ padding: 16 }, [TextView({ text: 'Item 1' })]),
    GridParams({ columnSpan: 2 }, 
        Card({ padding: 16 }, [TextView({ text: 'Item 2 (spans 2 columns)' })])
    ),
    Card({ padding: 16 }, [TextView({ text: 'Item 3' })])
]);

ConstraintLayout

A constraint-based layout using position: relative/absolute for flexible positioning.

Usage

ConstraintLayout(props = {}, children = [])

Constraint Parameters (for children)

Use Constrained(constraints, child) to position children:

Constraint Type Description
top number/string Distance from top
right number/string Distance from right
bottom number/string Distance from bottom
left number/string Distance from left
centerHorizontally boolean Center horizontally in parent
centerVertically boolean Center vertically in parent
horizontalBias number 0-1 bias for horizontal centering
verticalBias number 0-1 bias for vertical centering
width string/number 'match_parent' | 'match_constraint' | value
height string/number 'match_parent' | 'match_constraint' | value

Example

const layout = ConstraintLayout({
    width: 'match_parent',
    height: 400,
    background: '#f5f5f5'
}, [
    Constrained({
        top: 16,
        left: 16
    }, TextView({ text: 'Top Left' })),
    
    Constrained({
        centerHorizontally: true,
        centerVertically: true
    }, TextView({ text: 'Centered' })),
    
    Constrained({
        bottom: 16,
        right: 16
    }, MaterialButton({ text: 'Action' }))
]);

FrameLayout

An overlapping container where children can be positioned with gravity or absolute positioning.

Usage

FrameLayout(props = {}, children = [])

Frame Position Parameters (for children)

Use Positioned(positionProps, child) to position children:

Parameter Description
top, right, bottom, left Absolute positioning values
zIndex Stack order
gravity center | top | bottom | left | right | top_left | top_right | bottom_left | bottom_right

Example

const frame = FrameLayout({
    width: 400,
    height: 400,
    background: '#f5f5f5'
}, [
    // Background image
    Positioned({ gravity: 'center' },
        MaterialImage({ src: 'background.jpg', width: '100%' })
    ),
    
    // Centered content
    Positioned({ gravity: 'center', zIndex: 1 },
        Card({ padding: 24 }, [
            TextView({ text: 'Overlay Content' })
        ])
    ),
    
    // Bottom-right button
    Positioned({ gravity: 'bottom_right', bottom: 16, right: 16, zIndex: 2 },
        MaterialButton({ text: 'Action', variant: 'filled' })
    )
]);

LayoutParams (for LinearLayout/Column/Row)

LayoutParams(params, child)
Parameter Type Description

  • scrollToIndex(index)

Example

    data: initialData,
    adapter: (item, index) => {
        return Card({ padding: 16 }, [
            TextView({ text: item.title, textStyle: 'bold' }),
            TextView({ text: item.description, margin: { top: 4 } })
        ]);
    },
    layoutManager: 'linear',
    divider: true,
    onLoadMore: async (offset) => {

            data: data.items,
            hasMore: data.hasMore
        };
    }
});
